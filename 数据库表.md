# 数据库表结构设计文档

## 1. 用户表（tb_users）
| 字段名      | 数据类型 | 约束 / 默认          | 字段说明           |
| ----------- | -------- | -------------------- | ------------------ |
| user_id     | int      | 主键、自增、非空     | 用户ID             |
| telephone   | varchar  | 非空                 | 手机号             |
| password    | varchar  | 非空                 | 密码加密           |
| create_time | datetime | 非空                 | 注册时间           |
| update_time | datetime | 可空                 | 更新时间           |
| status      | int      | 非空，1=正常，0=禁用 | 账号状态           |
| is_deleted  | int      | 非空，默认0          | 0=未删除，1=已删除 |

------

| 字段名             | 数据类型 | 约束 / 默认                                            | 字段说明           |

| current_project_id | int      | 默认0                                                  | 当前项目           |
| create_time        | datetime | 非空                                                   | 注册时间           |
| update_time        | datetime | 可空                                                   | 更新时间           |
| status             | int      | 非空，1=正常，0=禁用                                   | 账号状态           |
| is_deleted         | int      | 非空，默认0                                            | 0=未删除，1=已删除 |
